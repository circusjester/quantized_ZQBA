
# Quantized ZQBA

This repository provides a quantized implementation of [ZQBA](https://github.com/Joana-Cabral/ZQBA) and serves as a codebase for follow-up research on ZKP.

## Datasets

The experiments in this repository use the following datasets:

- [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html)
- [CIFAR-100](https://www.cs.toronto.edu/~kriz/cifar.html)
- [Tiny ImageNet](http://cs231n.stanford.edu/tiny-imagenet-200.zip)
